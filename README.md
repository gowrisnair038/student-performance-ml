# Student Performance Analysis and Prediction

## Project Overview
This project analyzes student performance data and builds a Linear Regression model to predict exam scores.

## What I Did
- Cleaned the dataset
- Performed data analysis and visualization
- Built a Machine Learning model (Linear Regression)
- Evaluated model performance

## Features Used
- Study Hours
- Class Attendance
- Sleep Hours

## Model Performance
- R² Score (Test): 0.63
- The model explains 63% of the variation in exam scores.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Conclusion
Study hours and attendance have a positive impact on exam performance. The model provides a reasonably good prediction of student scores.
